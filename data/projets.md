## Projets de 1ère année (S1-S2)

### Analyse des AVC en France (Power BI & Python)
- **Description** : Développement d'un outil interactif pour évaluer l'étendue et les caractéristiques des cas d'AVC (Accident Vasculaire Cérébral) en France. L'outil identifie les facteurs de risque et de gravité pour aider à la prise de décision.
- **Technologies** : Power BI, Python
- **Compétences** : Analyse de données, datavisualisation, création de tableaux de bord

### Prédiction des prix d'appartements à Paris (Régression linéaire)
- **Description** : Application d'un modèle de régression linéaire pour prédire les prix des appartements à Paris en fonction de différents paramètres (surface, arrondissement, etc.)
- **Technologies** : R, Excel
- **Compétences** : Modélisation statistique, régression linéaire, analyse prédictive

### Traitement de données JSON et CSV (Python)
- **Description** : Projet technique axé sur le traitement et la manipulation de fichiers de données JSON et CSV avec des scripts Python
- **Technologies** : Python, JSON, CSV, Pandas
- **Compétences** : Programmation Python, manipulation de données, formats de fichiers

### Portfolio numérique (SAÉ)
- **Description** : Création d'un portfolio digital pour présenter mes travaux académiques et mes compétences
- **Technologies** : HTML, CSS, JavaScript
- **Compétences** : Développement web front-end, design responsive

### Présentation en anglais (Oral)
- **Description** : Présentation orale réalisée entièrement en anglais sur une destination touristique
- **Compétences** : Anglais professionnel, prise de parole en public, communication
